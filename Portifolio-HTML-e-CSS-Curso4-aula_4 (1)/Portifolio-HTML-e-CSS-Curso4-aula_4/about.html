Sobre mim

Sou André Azevedo de Oliveira, um desenvolvedor de software apaixonado por tecnologia e inovação. Ao longo da minha carreira, tive a oportunidade de trabalhar com diversas tecnologias, tanto no front-end quanto no back-end.

Tenho experiência com HTML, CSS, JavaScript, React, Angular, C#, Java SE e EE, Python com Django, além de bancos de dados como MySQL, PostgreSQL e SQL.

Meu foco está sempre em criar soluções eficientes e de alta qualidade, com ênfase em uma boa experiência de usuário. Adoro desafios e estou sempre em busca de aprender novas ferramentas e melhores práticas para aprimorar meus projetos.

Acredito que a combinação de boas práticas de desenvolvimento e inovação constante é o que torna um projeto verdadeiramente bem-sucedido.
